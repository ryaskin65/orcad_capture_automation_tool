# OrCAD Automation Tool

## Description
Python GUI with TCL scripts to automate routine tasks in OrCAD Capture schematics. Automates text search/replace, connector direction changes, and cable drawing. Scripts run via OrCAD command window through the Python app, saving engineers time on repetitive operations.

## Features
- Search and replace text in schematics
- Change connector (offPage) directions (input/output)
- Draw cables automatically according to the Exel table
- Run TCL scripts via Python GUI
